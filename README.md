# ExxonMobil DCF Valuation Model

## Overview
DCF model forecasting ExxonMobil's intrinsic value based on revenue projections, 
operating margins, and WACC discounting.

## Key Assumptions
- Revenue decline 2% annually for 3 years (energy transition + EV adoption acceleration)
- Operating margins decline to ~9.8% then recover to ~10.1%
- Terminal growth: 0.5% (mature industry)
- WACC: 9.2%

## Result
Intrinsic value: $53/share vs. current price ~$146

## Interpretation
Model assumes conservative revenue headwinds. Current price implies market expects 
stronger recovery or higher long-term growth. Sensitivity analysis (not yet included) 
would test alternative scenarios.
